# San Francisco Crime Data Analysis

## Overview
This project analyzes public crime data from San Francisco to identify patterns across time and location. 
The goal is to uncover trends, highlight high-risk areas, and communicate insights through data visualization.

## Tech Stack
- Python (Pandas, NumPy)
- SQL
- Tableau
- Jupyter Notebook

## Dataset
- Source: San Francisco Open Data Portal
- Contains records of reported crimes, including date, category, and location

## Project Workflow

### Data Cleaning & Preparation
- Removed missing and inconsistent values
- Standardized date and time formats
- Filtered relevant crime categories

### Exploratory Data Analysis
- Analyzed crime distribution by district
- Identified trends over time using SQL queries

### Visualization
- Built dashboards in Tableau
- Created a heatmap to show geographic crime clusters

## Key Insights
- Theft-related crimes peak during summer months
- High-traffic areas show consistently higher crime rates
- Certain districts emerge as recurring hotspots
